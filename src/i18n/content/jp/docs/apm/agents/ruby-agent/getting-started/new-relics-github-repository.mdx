New Relic は、Ruby リポジトリを GitHub の [newrelic-ruby-agent](https://github.com/newrelic/newrelic-ruby-agent/tree/main) にクローンしています。また、GitHub は限定的な rdoc 生成ドキュメントをホストしています: [rdoc.info/github/newrelic/newrelic-ruby-agent](https://rubydoc.info/github/newrelic/newrelic-ruby-agent).

main の head には通常、New Relic の最新リリースが含まれます。ただし、エッジをマスターにプッシュする機能は留保しています。GitHub からリリースをダウンロードする場合は、適切なタグを使用してください。通常、リリースのプレビュー バージョンをワーキング ブランチにプッシュしてから、一般提供のタグを付けます。

また、アイデアやバグフィックスについては、 [support.newrelic.com](https://support.newrelic.com/) まで、プルリクエスト（ブランチのみ）をお寄せください。
