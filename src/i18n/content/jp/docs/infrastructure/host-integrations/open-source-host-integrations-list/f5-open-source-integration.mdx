F5 [オープンソース統合は、](/docs/integrations/new-relic-integrations/getting-started/introduction-infrastructure-integrations) [F5](https://support.f5.com/csp/knowledge-center/software/BIG-IP) からのパフォーマンス メトリクス (仮想サーバー、プール、ノード、メモリ統計を含む) を監視し、データを New Relic に報告します。

要件、インストール、構成情報については、GitHub の [F5 オンホスト統合 README を](https://github.com/newrelic/nri-f5) 参照してください。

<Callout variant="important">
  New Relic は、この技術のモニタリングを可能にするために、この統合をオープンソース化しました。この統合は現状のまま提供され、保証やサポートはありませんが、GitHub を通じて問題の報告やこの統合への貢献が可能です。この統合のサポートは、 [Expert Services のサブスクリプションでご利用いただけます。](https://www.newrelic.com/expertservices) 。
</Callout>
